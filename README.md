# Women's Day Gift Clone

Static clone of the DearGift Women's Day page.

## Files

- `index.html`: main gift page.
- `customize.html`: browser editor for changing text, images, letter content, and music.
- `assets/app-config.json`: deploy-time configuration file.
- `assets/`: images, fonts, and visual assets.
- `musics/`: local background music.

## Customize

Open `customize.html`, edit the fields, then use **Lưu & xem**.

For permanent deployment changes, export JSON from `customize.html` and replace `assets/app-config.json` with that file.
